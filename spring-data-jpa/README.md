### UserService
* Регистрация новых пользователей
* Аутентификация пользователей и выдача JWT-токена
* Обновление и получение информации о профиле пользователя
* Управление ролями и правами доступа пользователей

### AuthorService
* Добавление, редактирование и удаление информации об авторе
* Получение списка авторов
* Получение детальной информации об авторе

### BookService
* Добавление, редактирование и удаление информации о книге
* Получение детальной информации о книге
* Поиск книг по различным критериям
* Работа с каталогом книг

### OrderService
* Формирование заказа на книги
* Отслеживание статуса заказа
* Получение истории заказов пользователя
* Расчет стоимости заказа

Каждый сервис производит валидацию JWT-токена, извлекает из него информацию о пользователе для определения его прав
доступа
